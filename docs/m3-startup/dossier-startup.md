# SLEIPNIR - Pitch Deck

## Slide 1: Cover

**SLEIPNIR**
*L'IA au service de l'automobile au Maroc*

> Intelligence artificielle. Recherche naturelle. Marché transparent.

---

## Slide 2: Problem

**Le problème des acheteurs de voitures au Maroc**

- **Données fragmentées** : les annonces sont éparpillées sur Avito, Moteur.ma, Facebook, Wandaloo
- **Pas de recherche en langage naturel** : les utilisateurs doivent connaître les filtres techniques
- **Pas de système de réputation** : aucun baromètre de fiabilité pour les vendeurs
- **Barrière linguistique** : le darija n'est supporté par aucune plateforme
- **Temps perdu** : des heures à comparer manuellement des dizaines d'annonces

> L'acheteur moyen passe **15-20 heures** avant de trouver la bonne voiture.

---

## Slide 3: Solution

**SLEIPNIR : La recherche intelligente automobile**

- **Recherche en langage naturel** : "Je cherche une SUV fiable, moins de 100 000 km, budget 150 000 DH"
- **Matching IA** : algorithme TOPSIS qui classe les meilleures options
- **Baromètre de réputation** : score de fiabilité basé sur l'historique du vendeur
- **Multi-sources** : agrège Avito, Moteur.ma, Facebook en un seul endroit
- **Interface mobile-first** : conçu pour le smartphone, premier point de contact

> SLEIPNIR transforme la recherche en une conversation intelligente.

## Slide 6: Technology

**Stack technique**

- **Frontend** : Next.js 14, TypeScript, Tailwind CSS
- **Backend** : Node.js, API REST
- **NLP** : Modèle de traitement du darija marocain
- **Moteur de décision** : TOPSIS (Technique for Order of Preference by Similarity to Ideal Solution)
- **Scraping** : Agrégation multi-sources (Avito, Moteur.ma, Facebook)
- **Base de données** : PostgreSQL, Redis cache
- **Déploiement** : Docker, CI/CD GitHub Actions

> Architecture scalable de 0 à 1M utilisateurs.

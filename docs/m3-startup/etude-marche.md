## Acteurs du marché digital

### 1. Avito.ma
- **Position** : Leader incontesté des petites annonces
- **Part de marché** : ~60% des annonces automobiles
- **Forces** : Base utilisateurs massive, brand awareness
- **Faiblesses** : Pas d'IA, pas de filtrage intelligent, réputation inexistante
- **Revenue model** : Annonces payantes

### 2. Moteur.ma
- **Position** : Spécialiste automobile
- **Part de marché** : ~15%
- **Forces** : Données techniques riches, comparateur
- **Faiblesses** : Interface datée, pas de NLP, une seule source
- **Revenue model** : Publicité, leads

### 3. Wandaloo.com
- **Position** : Forum communautaire
- **Part de marché** : ~10%
- **Forces** : Communauté active, avis utilisateurs
- **Faiblesses** : Non structuré, pas de recherche avancée
- **Revenue model** : Publicité

### 4. Facebook Marketplace
- **Position** : Alternative gratuite
- **Part de marché** : ~8%
- **Forces** : Intégration sociale, gratuit
- **Faiblesses** : Pas de filtres, arnaques fréquentes, pas d'historique
- **Revenue model** : N/A (écosystème Meta)

### 5. Concessionnaires en ligne
- **Part de marché** : ~5%
- **Forces** : Garantie, SAV
- **Faiblesses** : Prix élevés, choix limité

## Douleurs identifiées (Voice of Customer)

### Méthodologie
- 15 interviews utilisateurs (acheteurs potentiels)
- 5 interviews vendeurs/garagistes
- Analyse 200+ commentaires forums

### Top 5 Pain Points

| Rang | Douleur | Fréquence |
|------|---------|-----------|
| 1 | Informations dispersées, temps perdu | 92% |
| 2 | Pas de fiabilité/transparence vendeur | 85% |
| 3 | Difficulté à comparer les prix réels | 78% |
| 4 | Barrière linguistique (darija vs français) | 72% |
| 5 | Pas de recommandations personnalisées | 68% |

## Taille de marché

### TAM (Total Addressable Market)
280 000 transactions/an x 500 DH (valeur moyenne service) = **140M DH** (~$14M)

### SAM (Serviceable Available Market)
42 000 acheteurs digitaux x 500 DH = **21M DH** (~$2.1M)

## Taille de l'opportunité

### Pourquoi maintenant ?

1. **Maturité du NLP** : les modèles de langage sont enfin capables de comprendre le darija
2. **Adoption mobile** : smartphone = 78% des accès internet au Maroc
3. **Frustration existante** : le marché est mûr pour une solution IA
4. **Absence de leader** : aucun acteur ne domine la recherche intelligente
5. **Régulation** : le Maroc encourage l'innovation digitale (Plan Maroc Digital 2030)

### Timing

- **Fenêtre d'opportunité** : 18-24 mois avant qu'un acteur existant ne copie
- **Barrières à l'entrée** : Data proprietary, modèle NLP entraîné, réseau vendeurs
- **Effet réseau** : Plus d'utilisateurs = meilleurs résultats = plus d'utilisateurs

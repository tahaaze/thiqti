# SLEIPNIR - Modèle Financier

## Vue d'ensemble

Projections financières sur 3 ans pour SLEIPNIR, plateforme d'intelligence artificielle automobile au Maroc.

---

## Hypothèses clés

- **Pénetration digitale automobile Maroc** : 15% → 25% en 3 ans
- **Croissance YoY recherches digitales** : 20%
- **Taux de conversion gratuit → premium** : 5%
- **Prix premium** : 99 DH/mois (~$10)
- **Prix Pro (garages)** : 499 DH/mois (~$50)
- **Commission marketplace** : 3% sur transactions

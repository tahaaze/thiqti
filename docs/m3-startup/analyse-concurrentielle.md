## Analyse par acteur

### Avito.ma

**Forces :**
- Leader incontesté (~60% parts annonces auto)
- Base utilisateurs massive (5M+ utilisateurs/month)
- Brand awareness élevée au Maroc
- Modèle économique éprouvé

**Faiblesses :**
- Aucune intelligence artificielle
- Pas de filtrage sémantique
- Réputation vendeur inexistante
- Expérience utilisateur basique
- Fraudes et arnaques fréquentes

**Stratégie probable :**
- Ajout progressif de filtres IA
- Partenariats avec vendeurs professionnels
- Risque : trop lente pour innov

### Moteur.ma

**Forces :**
- Données techniques riches
- Spécialiste automobile
- Base données véhicules complète
- Comparateur intégré

**Faiblesses :**
- Interface ancienne (design 2010)
- Pas de NLP
- Une seule source (pas d'agrégation)
- Audience limitée
- Monétisation agressive (publicités)

**Stratégie probable :**
- Modernisation UI
- Intégration de filtres IA
- Risque : manque de ressources

### Wandaloo.com

**Forces :**
- Communauté active et engagée
- Avis utilisateurs authentiques
- Forums thématiques
- Contenu UGC riche

**Faiblesses :**
- Non structuré pour la recherche
- Pas de comparaison automatique
- Interface forum datée
- Mobile très limité
- Pas de modération efficace

**Stratégie probable :**
- Reste niche communautaire
- Pas de threat sérieux
- Peut devenir source de données

### Facebook Marketplace

**Forces :**
- Intégration écosystème social
- Gratuit
- Large audience
- Partage social intégré

**Faiblesses :**
- Pas de filtres automobiles
- Arnaques fréquentes
- Pas d'historique
- Pas de réputation fiable
- Pas de données techniques

**Stratégie probable :**
- Amélioration progressive des filtres
- Risque : peut devenir un concurrent direct si investissement auto
